# Deaguin

Um jogo roguelike 2D de geração procedural
onde você é um pinguim que se perdeu de uma
geleira que se partiu e está tentando retornar para casa.

## Fases

 O jogo contém 3 fases sendo, praia, cidade e geleira

### fases prontas

- [x] Praia
- [ ] Cidade
- [ ]  Geleira
